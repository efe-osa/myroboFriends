# roboFriends
Robofriends is a web application that displays robots accessed from the Robohash API and gives users the ability to search through the robots list for a robots of their liking.

## Technologies
React, Redux

## Getting Started
- Clone repo
- cd myroboFriends 
- npm install
- npm run start
